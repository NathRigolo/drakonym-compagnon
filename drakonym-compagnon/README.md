# Drakonym Compagnon

Une application web mobile-first pour gérer ses fiches de personnage du jeu de rôle **Drakonym** *(Core Rulebook v1.0)*.

🎲 **App en ligne** : https://nathrigolo.github.io/drakonym-compagnon/

---

## Fonctionnalités

- **Multi-fiches** — Crée, duplique, renomme, supprime plusieurs personnages. Bascule entre eux en un tap.
- **Fiche complète** — Identité, attributs, pools (HP, Mana, Grit, Défense), Wounds avec overflow, statuses officiels.
- **Capacités** — Perks, Sorts (avec calcul automatique du coût en succès via la Mana Cost Table), Techniques, Wild Perks. Color picker et tri par couleur.
- **Équipement** — Armes, armures et outils. 21 presets d'armes et 6 presets d'armures officiels. Mode attaque dans le lanceur de dés.
- **Dragon** — Identité, attributs, Points de Lien, Souffle (avec gestion des charges), Pillars, Bond Perks et équipement dragon (11 armes + 9 armures officielles).
- **Lanceur de dés** — Gère Faveur/Fardeau, Shadow Die, mode attaque, modificateurs, Hero Points (Faveur, Reroll fine, Aide allié), historique des 20 derniers jets, presets de jets sauvegardés.
- **Histoire & Notes** — Apparence, background, liens, notes libres par personnage.
- **Import/Export JSON** — Sauvegarde tes fiches, transfère-les entre ton PC et ton téléphone.
- **PWA installable** — Fonctionne hors-ligne après le premier chargement.
- **Bilingue technique** — Interface en français, termes du jeu (Mana, Grit, Wild Perks, etc.) gardés en VO pour rester aligné avec le rulebook.

## Installation

Aucune installation requise — l'app fonctionne directement dans ton navigateur. Sur mobile, tu peux l'installer comme une vraie app via le menu "Ajouter à l'écran d'accueil" (Chrome / Safari).

## Stockage des données

Toutes tes fiches sont stockées **localement** dans le navigateur (`localStorage`). Rien n'est envoyé sur internet. Pour transférer entre devices, utilise l'export/import JSON dans l'onglet **Plus**.
